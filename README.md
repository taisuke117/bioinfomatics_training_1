## 1. 実習の目的

本実習では、プログラミング言語 **Python** の基礎を学び、コンピュータを用いたデータ解析の感覚を掴むことを目的とします。

将来的なバイオインフォマティクス解析に向けた準備として、まずは「プログラムでルールを作る」「データを図にする」という基本操作を習得しましょう。

## 3. 課題１：条件分岐とループの理解（ジャンケンから相補鎖作成へ）

### 【1-1】ジャンケンプログラムの作成

以下の機能を備えたジャンケンプログラムを作成してください。

1. ユーザーが「グー・チョキ・パー」のいずれかを入力する
2. コンピュータがランダムに手を選ぶ
3. 勝ち・負け・あいこを判定し、結果を表示する
4. **【発展】** 負けた時にコンピュータが捨てセリフを複数個ランダムで吐くように工夫してみましょう

### 【1-2】バイオへの応用：DNA相補鎖作成

ジャンケンの「AならBを出す」という条件分岐の考え方を使って、DNA配列を入力すると相補鎖を返すプログラムを作成してください。

```
入力例： ATGCTTAG
出力例： TACGAATC
```

変換ルール：`A → T` / `T → A` / `G → C` / `C → G`

## 4. 課題２：データの可視化（身近なデータの分析）

バイオの大きなデータ（発現量など）を扱う前に、身近なデータでグラフ作成の練習をします。

### 【2-1】データの準備

以下のいずれかのデータを準備し、CSVまたはリスト形式にまとめてください。

- コンビニ・カフェ（スタバ等）のメニュー数種類の **カロリー・タンパク質・脂質**
- スマホの **歩数データ** や **スクリーンタイム**（1週間分）
- その他、自分で興味のある数値データ

### 【2-2】グラフによる可視化

Pythonのライブラリ（`pandas`, `matplotlib` / `seaborn`）などを用いて、以下の2種類のグラフを作成してください。

| グラフ種類 | 目的 | 例 |
|---|---|---|
| 棒グラフ | 項目の比較 | メニューごとのカロリー比較 |
| 散布図 | 2変数の関係 | タンパク質 vs 脂質 |

## 6. 自走するためのルール（質問の前に）

エラーが出て詰まった場合は、**以下の順番で** 解決を試みてください。

```
Step 1. エラー文をよく読む
        └─ どこで間違っているかヒントが書いてあります

Step 2. AIに聞く
        └─ エラー文をそのままコピーして貼り付け、
           「このエラーを解消するコードと、
            なぜエラーになったかの解説を教えて」と聞く

Step 3. 教員への質問
        └─「何を試して、どこで詰まっているか」を
           添えて質問に来てください
```

> 「動かない → すぐ聞く」ではなく、Step 1を必ず経て、何が悪いのか予想してみましょう。
> エラーと格闘する時間こそが、最大の学習機会です。
